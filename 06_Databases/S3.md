# S3

`S3` — API object storage, изначально Amazon S3, но совместимые реализации есть у разных провайдеров.

---

# Базовые команды

```bash
aws s3 ls s3://bucket/
aws s3 cp file.txt s3://bucket/path/file.txt
aws s3 sync ./backup s3://bucket/backup/
aws s3api head-object --bucket bucket --key path/file.txt
```

---

# Что важно

```text
bucket policy
IAM permissions
versioning
server-side encryption
lifecycle rules
object lock для critical backups
access logs/audit
```

---

# Частые проблемы

```text
AccessDenied
wrong region/endpoint
KMS key policy запрещает decrypt
public access случайно открыт
дорогие list operations
```
